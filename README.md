# CeCe Typing Game

Mini game tap danh chu cho tre em voi chu ca CeCe trong dai duong.

- Phim mui ten: dieu khien CeCe.
- Enter: dop bong bong gan nhat.
- Go dung ky tu trong bong bong de CeCe lon len.
- Sai 3 lan thi CeCe nho lai va nha bong bong ra.

Choi online: https://huynhcokho.github.io/CeCe/
